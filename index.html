Electricube is my most sophisticated game project, which I am developing solo, it is a sokoban style puzzle game, where the player needs to provide electricity to the rotating block in order to win the level, there are other mechanics in the game which get activated by electricity, being the heart of puzzles in this game.

  • Levithorn (Group project)

Github link: https://github.com/arshil0/Levithorn

Levithron is a 2D puzzle platformer that me and my classmates worked on for our Game Development class, it’s set out in a mystical forest where blocks have their own gravity rules, the player has the ability to manipulate those blocks in order to solve puzzles or use them as jumping platforms.

I was responsible for the Game / Level Design, programming the mechanics and a level editor, and making the music / sound effects for the game, this game wouldn’t have looked nearly as good without the talent of my teammates!

Levithorn was developed (and still getting new levels) using Unity and C#

  • AI Infection

Game link: https://melonkng.itch.io/ai-infection

(Apologies for the laggy video, I couldn’t record the game smoothly)

AI Infection was my first game that I put out on the internet, in 2023, this was a game jam game for the GMTK game jam, where developers had 48 hours to make a game, the theme was Roles Reversed.

In this game, the player starts off by shooting incoming enemies, but then on the next wave of enemies the roles swap, you get controlled by an AI and your job is to spawn the enemies to try and lose the game for the AI, after every wave, the player chooses between 3 upgrades (or downgrades) to try and gain an advantage while also not giving too much to the AI, as it also receives the same upgrades.

This game was made using Godot.
